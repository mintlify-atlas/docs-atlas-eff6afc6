Alpamayo 1.5 is a 10-billion-parameter Vision-Language-Action (VLA) model from NVIDIA that supercharges autonomous driving research with interactive, steerable reasoning. It combines a Qwen3-VL visual language backbone with a diffusion-based action expert to generate interpretable chain-of-causation reasoning traces alongside precise 6.4-second trajectory predictions — all conditioned on multi-camera video inputs.

## Getting started

<Steps>
  <Step title="Install the environment">
    Set up a Python 3.12 virtual environment with `uv` and install all dependencies including Flash Attention.

    ```bash
    uv venv a1_5_venv
    source a1_5_venv/bin/activate
    uv sync --active
    ```
  </Step>
  <Step title="Authenticate with HuggingFace">
    Request access to the gated model and dataset, then authenticate with your HuggingFace token.

    ```bash
    hf auth login
    ```
  </Step>
  <Step title="Run the test script">
    Download example data and model weights, then run inference to verify your setup.

    ```bash
    python src/alpamayo1_5/test_inference.py
    ```
  </Step>
  <Step title="Explore the notebooks">
    Dive deeper with the provided Jupyter notebooks covering navigation guidance, VQA, and multi-camera inference under `notebooks/`.
  </Step>
</Steps>

<Note>
  Alpamayo 1.5 requires at minimum **24 GB VRAM** for single-sample inference. The model weights are ~22 GB and are downloaded automatically on first run. This model is for **research use only** — not for commercial applications.
</Note>
